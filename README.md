# WindowsBar
A simple Windows Bar

## Minimum Requirements:
- Windows 11
- Rainmeter 4.5.23

## Features
- Scalable UI
- SMTC support
- WebNowPlaying support

## Installation
1 - Download the latest version from [Releases](https://github.com/Meti0X7CB/WindowsBar/releases) tab.

2 - Install the skin by double-clicking the .rmskin file and follow the steps in the installer.
